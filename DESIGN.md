# Frontend 디자인 명세

## 스택
- Next.js 15 (App Router)
- Tailwind CSS
- shadcn/ui (컴포넌트)
- lucide-react (아이콘)
- react-markdown (리포트 렌더링)

## 디자인 방향
- 콘텐츠 중심, 깔끔, 읽기 편한 레이아웃
- Morning Brew / Substack 스타일
- 최대 너비 768px, 중앙 정렬

## 화면

### / (구독 페이지)
```
┌─────────────────────────────┐
│  📊 뉴스 브리핑              │
│  매일 아침, 맞춤 뉴스        │
│                             │
│  이메일  [              ]    │
│  이름    [              ]    │
│                             │
│  관심 분야                   │
│  ☑ 매크로   ☑ 금융   ☐ 테크  │
│  ☐ AI      ☐ 에너지  ☐ 부동산│
│  ☐ 정치    ☐ 스타트업         │
│                             │
│  [구독하기]                  │
└─────────────────────────────┘
```
- shadcn/ui: Input, Checkbox, Button
- 구독 성공 시 토스트 메시지

### /reports (리포트 목록)
```
┌─────────────────────────────┐
│  ┌───────────────────────┐  │
│  │ 📊 4월 13일 브리핑     │  │
│  │ 연준 동결, 삼성 실적.. │  │
│  │ 기사 7개 · 3분 전      │  │
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ 📊 4월 12일 브리핑     │  │
│  │ ...                   │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```
- shadcn/ui: Card
- 날짜 내림차순
- 클릭하면 /reports/[id]로 이동

### /reports/[id] (리포트 상세)
```
┌─────────────────────────────┐
│  경제 브리핑 — 4월 13일      │
│                             │
│  오늘의 핵심                 │
│  > 연준이 기준금리를 동결...  │
│                             │
│  ─────────────────────────  │
│  [HIGH] 연준 기준금리 동결   │
│  Reuters · 원문 보기         │
│  번역된 본문...              │
│  ─────────────────────────  │
│  [MEDIUM] 삼성전자 실적      │
│  한경 · 원문 보기            │
└─────────────────────────────┘
```
- react-markdown으로 content_md 렌더링
- 영향도 태그 색상: HIGH(빨강), MEDIUM(노랑), LOW(회색)
- 원문 링크는 새 탭

### /unsubscribe/[token] (구독 해지)
```
┌─────────────────────────────┐
│  구독을 해지하시겠습니까?     │
│  [해지하기]  [돌아가기]       │
└─────────────────────────────┘
```
- 간단한 확인 페이지

## 컴포넌트 목록

```
components/
├── SubscribeForm.tsx    # 이메일 + 이름 + 섹터 선택 폼

```
